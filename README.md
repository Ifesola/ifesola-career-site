# Ifesola Fadare – Web Portfolio

This is a personal portfolio site built for my Web Development Foundations course.  
It showcases my résumé, cover letter, and career goals using semantic HTML, CSS, and a bit of JavaScript (dark mode toggle and dynamic dates).

## Project Structure

```text
index.html          # Résumé page (home)
cover-letter.html   # Cover letter
career-goals.html   # Career goals, audio, and video
style.css           # Global styling for all pages
images/             # Profile photo and other images
audio/              # Intro audio clip
video/              # Career goals video
docs/               # Downloadable résumé files (PDF/DOCX)

Features

Semantic HTML5 structure (header, nav, main, section, article, footer)

Global navigation between résumé, cover letter, and career goals

Responsive layout with profile header and two-column résumé sections

Dark mode toggle using JavaScript + CSS

Auto-updating dates from JavaScript (résumé + cover letter)

Audio and video embedded on the Career Goals page

Downloadable résumé as PDF
